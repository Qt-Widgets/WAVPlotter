## Build from source
 Install dependencies
  
    sudo apt install qt5-qmake qtmultimedia5-dev qtbase5-dev libgl1-mesa-dev

Clone this repository

    git clone https://github.com/ThePBone/WAVPlotter

Compile sources

    cd WAVPlotter
    qmake
    make
    
You should now be able to execute it:

    ./WAVplotter
